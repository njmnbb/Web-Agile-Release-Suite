# Web-Agile-Release-Suite

## Getting Started
1. Download and install either [Visual Studio Code](https://code.visualstudio.com/Download) or WebStorm (by requesting it via Request Central)
2. Download and install Node.js and NPM from [Node's website](https://nodejs.org/en/)
3. After Node.js installs, open the built-in terminal on OSX or by using the terminal bundled with Visual Studio Code/WebStorm and check to see if Node.js and NPM have properly installed. Your version should be 3.0.0 or higher: 
```
npm -v
```
4. Run the following command to install TypeScript:
```
npm install -g typescript
```
5. Run the following command to install the Angular CLI: 
```
npm install -g @angular/cli@1.0.0
```
6. Verify that the Angular CLI has installed properly:
```
ng --version
```
7. Create a new directory and navigate to it:
```
mkdir web-agile-release-suite
cd web-agile-release-suite/
```
8. Clone this GitHub repo into your newly created directory:
```
git clone https://github.com/njmnbb/Web-Agile-Release-Suite.git
```
